Session menu


The ProCope GUI can store current session information to the file system. That means that all objects you loaded into the program can be restored later on by loading the corresponding session file. The following objects will be stored when saving a session:
  • protein networks
  • complex sets
  • purification datasets
  • GO settings (file is linked, see note below)
  • localization data (file is linked, see note below)
  • protein annotations
  • loaded name mappings
Note: All objects except GO settings and localization data are completely stored in the session file. That means you could delete or move the original datafiles and the session would restore successfully nevertheless.

Load session

Loads a session from a file.

Save session

Save current session to GZIPed file.

Exit

Exits the ProCope graphical user interface.
