Purification data menu


Contains methods to load, save and manipulate purification data sets.

Overview

  • Load purification data from file
  • Save purification data to file
  • Export to Cytoscape

  • Rename dataset
  • Dispose dataset

  • Calculate scores
  • Merge
  • Get bait-prey interactions

Load purification data from file

Loads a set of purifications from a user-specified file.

See also: File formats

Save purification data to file

Saves the selected purification data set to a user-specified file.

See also: File formats

Export to Cytoscape

Exports a Cytoscape-compatible XGMML file or directly creates a new graph within Cytoscape when running in plugin mode.

See also: Cytoscape integration

Rename dataset

Assigns a new internal name for the selected dataset.

Dispose dataset

Deletes the purification dataset from the list. Note: This will not affect an eventual file the dataset was loaded from.

Calculate scores

Calculates a PPI scores network from the selected purification data set.

See also: Score network generation

Merge

Merges the selected purification datasets. The experiment lists of all involved datasets are simply concatenated.

Get bait-prey interactions

Creates a directed network which contains one edge for each bait-prey interaction in the selected purification data sets.
