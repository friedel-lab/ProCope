Installation guide

This page contains information about how to get ProCope running on your system.

Extracting ProCope

The ProCope package is provided as both a gzipped tar and a zip archive.
  • procope-X.Y.tar.gz
  • procope-X.Y.zip
Where X.Y is the version number of the ProCope package you downloaded.

Linux

Use a graphical tool (e.g. File Roller) or unpack the archive into a directory of your choice from the command line:

tar xvzf procope-X.Y.tar.gz
or
unzip procope-X.Y.zip

Note that this command will create a sub-directory procope-X.Y in the current working directory.

Windows

Use any archiving tool (e.g. 7-Zip) to unpack one of the archives (.zip or .tar.gz) into a directory of your choice.

Starting ProCope

Depending on which part of ProCope you want to use, please read the respective pages of the documentation:
  • Graphical user interface
  • Command line tools
  • Java API
